## 📦 How to Deploy (5 minutes)

### Step 1 — Fork or clone this repo
```bash
git clone https://github.com/reachgikz-netizen/father-s-seo-content-audit.git
cd father-s-seo-content-audit
```

### Step 2 — Push to GitHub
```bash
git add .
git commit -m "Deploy SEO Audit Tool"
git push origin main
```

### Step 3 — Enable GitHub Pages
1. Go to your repo → **Settings** → **Pages**
2. Under **Source**, select `main` branch and `/ (root)` folder
3. Click **Save**
4. Your tool is live at `https://reachgikz-netizen.github.io/father-s-seo-content-audit` 🎉

---

## 🔧 How to Use
1. Open the live URL
2. Upload one or more `.docx` files
3. Enter your target keyword
4. Click **Run SEO Audit**
5. Download the Excel report
